Usage: mixcr exportPreprocTables [--chains <chain>]... [--no-warnings] [--verbose] [--help] pa.json
                                 [.gz] table.(tsv|csv)
Export preprocessing summary tables.
      pa.json[.gz]         Input file with postanalysis results.
      table.(tsv|csv)      Path for output file.
      --chains <chain>     Export only for specified immunological chains.
      -nw, --no-warnings   Suppress all warning messages.
      --verbose            Verbose messages.
  -h, --help               Show this help message and exit.
